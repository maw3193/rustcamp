CI PLAN:

* Check it actually has no errors `cargo check --all`
* Check it's all formatted correctly `cargo fmt --all --check` (tbh I want a pre-commit hook for this, but couldn't get pre-commit to work)
* Run all tests `cargo test --all`
* Generate all documentation `cargo doc --all`
* Satisfy the linter `cargo clippy`
* Check coverage `cargo tarpaulin --all`

```
on: push
name: Clippy check

# Make sure CI fails on all warnings, including Clippy lints
env:
  RUSTFLAGS: "-Dwarnings"

jobs:
  clippy_check:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Run Clippy
        run: cargo clippy --all-targets --all-features
```
